# Construction Project Risk Analysis Dashboard

## Overview
An interactive Power BI dashboard analyzing risk, cost, and resource allocation 
across 1,300 construction project tasks.

## Key Metrics
- Total Tasks: 1,300
- Total Material Cost: $64.38M
- Average Task Duration: 43 days

## Dashboard Pages

### Page 1 — Risk Analysis
- Risk level distribution across all tasks (Low/Medium/High)
- Material cost breakdown by risk level
- Task duration analysis by risk level
- Interactive slicer for dynamic filtering

### Page 2 — Labor & Equipment Analysis
- Labor demand by risk level
- Equipment utilization by risk level
- Resource vs site constraint scatter plot

## Tools Used
- Power BI Desktop
- Dataset: Construction Project Management Dataset (Kaggle)

## Key Insights
- 50% of tasks are low risk, but high risk tasks represent 20% of all work
- Low risk tasks drive the highest total cost due to volume
- Labor and equipment demand follow the same pattern across risk levels
